# Avalone — тексты для портала

## О чём портал

Avalone — это лёгкий каталог небольших приложений, которые делают одну вещь хорошо. Сейчас в экосистеме два приложения: **Counta** и **Routa**.

Каждое приложение — самостоятельный сервис со своим входом и базой данных. Avalone пока не является единым центром идентификации; SSO и общий аккаунт — направление на будущее.
